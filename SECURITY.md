# Security Policy

## Supported Versions
We currently only provide security updates for the latest version of the application.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
If you discover a security vulnerability within this project (e.g., issues with how data is stored or potential exploits), please do not open a public GitHub Issue. Instead, help us keep the community safe by following these steps:

1. **Email the developer:** Please send a detailed report to [abdulhadi09080700@gmail.com].
2. **Details to include:** - A description of the vulnerability.
   - Steps to reproduce the issue.
   - Potential impact.
3. **Response time:** I will acknowledge your report within 48-72 hours and provide a timeline for a fix.

## Privacy Note
This application records your screen and saves it locally to your machine. 
* **No data is ever sent to a remote server.** * **The developer has no access to your recordings.**
Users are responsible for ensuring they do not record sensitive information (passwords, private data) unintentionally.
